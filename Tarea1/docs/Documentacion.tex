\section*{\centering Descripcion del trabajo}

En este trabajo utilizamos \texttt{Babylon.js} y \texttt{Three.js} para crear una representacion visual en 3D de figuras geometricas.

Los requisitos a cubrir son:
\begin{itemize}
    \item Manejo de rezise.
    \item Relacion de aspecto y camara.
    \item 4 Figuras geometricas diferentes.
    \item Iluminacion.
    \item 2 Materiales diferentes.
    \item Animacion
    \item Rotacion, zoom, pan
\end{itemize}
